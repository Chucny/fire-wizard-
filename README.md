<h3>Fire Wizard</h3>
A 3D game where you fight a cat. This is kind of a test and Fire Wizard 2 is going to come out at some point.<br>
Controls:<br>
Left click: Shoot fireball <br>
WASD: Move<br>
O: exit<br>
